Comment ça marche ?

  1. 1ère étape

    Activer le websocket dans OBS

    Initialiser le websocket dans OBS pour permettre la communication avec le site web. Ajouter un mot de passe pour sécuriser la connexion.

  2. 2ème étape

    Connexion au websocket

    Ensuite, il suffit de renseigner l'adresse IP, le port et le mot de passe pour se connecter au websocket d'OBS.

  3. 3ème étape

    Utiliser les boutons

    Enfin, il ne reste plus qu'à utiliser les boutons pour contrôler les scènes, transitions et alertes de votre diffusion en direct.
